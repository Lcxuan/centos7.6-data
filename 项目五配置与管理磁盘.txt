fdisk		磁盘分区
			命令	功能
			a		调整硬盘启动分区
			d		删除硬盘分区
			l		列出所有支持的分区类型
			m		列出所有命令
			n		创建新分区
			p		列出硬盘分区表
			q		不保存更改，退出fdisk命令
			t		更改分区类型
			u		切换所显示的分区大小的单位
			w		把修改写入硬盘分区表，然后退出
			x		列出高级选项

mkfs		建立文件系统
			mkfs [参数] 文件系统
			参数：
				-t	指定要创建的文件系统类型
				-c	建立文件系统前首页检查坏块
				-l file 从文件file中读磁盘坏块，file文件一般是由磁盘检查程序产生的
				-V	输出建立文件系统详细信息

fsck		用于检查文件系统的正确性,并对Linux磁盘进行修复
			fsck [参数选项] 文件系统
			参数：
				-t		给定文件系统类型
				-s		一个一个地执行fsck命令进行检查
				-A		对/etc/fstab中所有列出来的分区进行检查
				-C		显示完整的检查进度
				-d		列出fsck的debug
				-P		在同时有-A选项时，多个fsck的检查一起执行
				-a		如果检查中发现错误，则自动修复
				-r		如果检查有错误，询问是否修复
			
df			用来查看文件系统的磁盘空间占用情况
			df [参数选项]
			参数:
				-a	显示所有文件系统磁盘使用情况，包括0块的文件系统
				-k	以k字节为单位显示
				-i	显示i节点信息
				-t	显示个指定类型的文件的磁盘空间使用情况
				-x	列出不是某一指定类型文件系统的磁盘空间使用情况
				-T	显示文件系统类型

du			用于显示磁盘空间的使用情况
			du [参数选项] [文件或目录名称]
			参数：
				-s	对每个name参数只给出占用的数据块总数
				-a	递归显示指定目录中个文件及子目录中个文件占用的数据块数
				-b	以字节为单位列出磁盘空间使用情况
				-k	以1024字节为单位列出磁盘空间情况
				-c	在统计后加上一个总计（系统默认设置）
				-l	计算所有文件大小，对硬链接文件重复计算
				-x	跳过在不痛文件系统上的目录，不予统计
				
mount		将文件系统挂载到系统
			mount 选项 设备 挂载点
			选项：
				-t	指定要挂载的文件系统的类型
				-r	如果不想修改要挂载的文件系统，可以使用该选项以只读方式挂载
				-w	以可写的方式挂载文件系统
				-a	挂载/etc/fstab文件中记录的设备
				
umount		卸载文件系统
			umount	设备 挂载点
			
自动挂载文件：/etc/fstab
	fs_spec	 fs_file  fs_vfstype  fs_mntops  fs_freq  fs_passno
	fs_spec：将要挂载的设备文件
	fs_file：文件系统的挂载点
	fs_vfstype：文件系统类型
	fs_mntops：挂载选项，决定传递给mount命令时如果挂载，个选项之间用逗号隔开
	fs_freq：由dump程序决定文件系统是否需要备份，0表示不备份，1表示备份
	fs_passno：由fsck程序决定引导时是否检查磁盘以及检查次序，取值可以为0，1
			
			
RAID	Redundant Array of IndexPensive Disks 独立磁盘冗余阵列
		将多个链家的小型磁盘驱动器合并成一个磁盘阵列，提高存储性能和容错功能
		RAID0	将多个磁盘合并成一个大的磁盘，不具有冗余，并行I/O，速度最快
		RAID1	把磁盘阵列中硬盘分为两组，互为镜像，当一个硬盘出现故障，可以使用镜像上的数据恢复，提高容错能力
		RAID3	以一个硬盘来存放数据的奇偶校验位，数据分段存储于其余硬盘中
		RAID5	向阵列中的磁盘写数据，奇偶校验数据存放在阵列中的各个盘，允许单个磁盘出错，以数据的校验位来保存数据的安全

		参数：
